This application serves as a note taking client intended for entering a set of notes quickly.

How to use
	- Enter your note in the field below
	- Press the "Save" button and it will appear in the list

	- Review notes by clicking on them from the table
	- Quickly make changes and save them 

Keyboard Shortcuts
	- CMD+S --> save or add a new note
	- DELETE --> deletes a note from list (if selected)
